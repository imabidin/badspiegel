Die ideale Lösung für eine saubere Außenansicht, wenn Sie die Helligkeit separat und stufenlos regeln möchten.

  • Der Drehdimmer wird an die Innenseite Ihres Schranks montiert und ist so jederzeit leicht erreichbar.
  • Kann auf Wunsch mit einer Steckdose oder einem USB-Anschluss im selben Rahmen kombiniert werden.
