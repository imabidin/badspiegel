Die perfekte Lösung für eine saubere Außenansicht, wenn ein separater Schalter für die Beleuchtung gewünscht wird.

  • Der Schalter wird an die Innenseite Ihres Schranks montiert und ist so jederzeit leicht erreichbar.
  • Kann auf Wunsch mit einer Steckdose oder einem USB-Anschluss im selben Rahmen kombiniert werden.
