Geben Sie Ihrem Spiegel den letzten Schliff: Mit hochwertigen Blenden verschließen Sie seitliche Ansichten und erzielen eine rundum ansprechende Optik.

Kunststoff lichtdurchlässig

Wer die Wand hinter dem Spiegel weiterhin beleuchten möchte, greift auf lichtdurchlässige Blenden zurück. Sie verdecken das LED-Band und sorgen für eine saubere Seitenansicht, ohne die Lichtausbeute zu beeinträchtigen.
